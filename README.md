# Wedding Invitation | Gautham & Deekshitha

An ultra-premium, cinematic wedding invitation website designed for the wedding of **Gautham and Deekshitha**.

## ✨ Features
- **Cinematic Entry**: A dramatic curtain-opening sequence that introduces the invitation.
- **GSAP Animations**: Fluid and modern animations throughout the user experience.
* **Ambient Effects**: Falling flower petals and floating particles to create a romantic atmosphere.
- **Interactive Story**: A beautiful section dedicated to the couple's story.
- **Detailed Timeline**: A chronological list of wedding events with location and time.
- **Integrated RSVP**: A seamless RSVP form powered by Formspree.
- **Background Music**: Integrated audio player with cinematic music.

## 🚀 Getting Started
To view the invitation locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/Harshithpalan/Wedding-invitation.git
   ```
2. Open `index.html` in your browser or run a simple local server:
   ```bash
   npx http-server
   ```
